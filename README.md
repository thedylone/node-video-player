# node-video-player
 
1. display videos

2. store and modify information of the videos

3. simple method of adding video to the database

4. client side modification of database

## directory

videos are stored in the following format:

```
videos/
└── source/
    └── title/
        ├── video1.mp4
        └── video2.mp4
```

for example:

```
hololive/
├── suisei/
│   ├── tetris/
│   │   ├── tetris1.mp4
│   │   └── tetris2.mp4
│   └── mario kart/
│       └── mario.mp4
└── pekora/
    └── minecraft/
        └── tnt.mp4
```
